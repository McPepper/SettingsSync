# SettingsSync

SettingsSync is a World of Warcraft addon that helps you quickly apply your preferred UI settings on any character.

## Features
- Apply curated settings with one click.
- Modular support for action bars, chat, edit mode profiles, and CVars.
- Designed for quick setup after a fresh install or on alts.

## Installation
1. Download or clone this addon into your WoW AddOns folder:
   `World of Warcraft\_retail_\Interface\AddOns\SettingsSync`
2. Ensure the folder name is exactly `SettingsSync`.
3. Launch the game and enable the addon in the AddOns list.
